# Fußball Training AI

Eine lokale Demo-App für Jugendfußballtrainer. Trainer wählen Altersklasse, Dauer, Schwerpunkt, Spieleranzahl, Niveau und Material und erhalten einen strukturierten Trainingsplan für den Platz.

## Features

- Trainingsplan mit Aufwärmen, zwei Hauptteilen, Spielform und Abschluss
- Inhalte passend zu Altersklasse und Trainingsschwerpunkt
- Coachingpunkte, Varianten und Material pro Abschnitt
- Trainer-Notiz für den Fokus der Einheit
- Text kopieren sowie Drucken oder als PDF speichern
- Mobile Darstellung für Smartphone und Tablet
- Formularprüfung für Dauer, Spieleranzahl und Material

## Lokale Installation

```bash
npm install
```

## Entwicklung starten

```bash
npm run dev
```

Die App läuft danach lokal, meist unter:

```text
http://localhost:5173/
```

## Produktionsbuild

```bash
npm run build
```

## Hinweis

Diese Demo läuft vollständig im Browser. Es gibt kein Backend, keine Anmeldung und keine Datenspeicherung.
